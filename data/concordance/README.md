# LOSO Concordance Results

Per-project Leave-One-Sample-Out (LOSO) differential expression and GSEA concordance results.

## Directory Structure

Each BioProject has:
- `<BioProject>_concordance.tsv` — per-sample concordance metrics (Spearman ρ, Jaccard, direction concordance)
- `<BioProject>/` — subdirectory with per-sample DE/GSEA intermediate outputs

## Example

See `PRJNA1014965/` for sample files. Full data available on Zenodo.
