# Bowtie2 Alignment Statistics

Per-SRR alignment statistics extracted from Bowtie2 stderr output.

## File Naming

`<SRR>_bowtie2_stats.tsv` — TSV with alignment rates, mapping counts, and concordant pair statistics across trimming modes.

## Example

See `PRJNA1014965/` for sample files. Full data available on Zenodo.
