# Utility Scripts

Ancillary helper scripts for specialized analyses.

## Sex Calling

Scripts for inferring biological sex from RNA-seq expression (XIST-based).

| Script | Purpose |
|--------|---------|
| `sexcall.py` | Sex inference from gene expression |
| `sexcall.sh` | Shell wrapper for sex calling |
| `extract_xist_bed.sh` | Extract XIST region from annotation |
| `install_sexcall.sh` | Environment setup for sex calling |
